# @thekzbn.write

A "bored" themed Markdown editor built with intention.

## Features

- **Distraction-free Writing**: Minimalist interface.
- **Live Preview**: See your markdown rendered in real-time.
- **Cloud Sync**: Sign in with Google or Email to save your work to the cloud.
- **Local History**: Basic undo/redo functionality.
- **Image Uploads**: Drag and drop or upload images directly to Firebase Storage.

## tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Vanilla CSS (Variables & Flexbox)
- **Backend/Auth**: Firebase (Auth, Firestore, Storage)

## Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Set up your `.env.local` with Firebase credentials.
4. Run `npm run dev`.
